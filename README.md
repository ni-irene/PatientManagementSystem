# Patient Management System (Java OOP + Exception Handling)

## 📌 Description
This project is a simple Patient Management System developed using Java. It demonstrates Object-Oriented Programming (OOP) principles and robust exception handling to simulate real-world hospital operations.

---

## 🚀 Features
- Add patients and doctors
- Create appointments
- Prevent double booking
- Validate working hours (09:00–17:00)
- User input with menu system
- Exception handling for errors

---

## 🧠 OOP Concepts Used
- Encapsulation
- Abstraction
- Inheritance
- Polymorphism

---

## ⚠️ Exception Handling
### Unchecked Exceptions:
- IllegalArgumentException
- InputMismatchException

### Custom Exceptions:
- InvalidTimeException
- DoctorUnavailableException

---

## ✅ Business Rules Enforced
- Doctors cannot be double booked at the same time
- Appointments must be within working hours
- Invalid inputs are handled gracefully

---

## ▶️ How to Run
1. Compile all `.java` files
2. Run `Main.java`
3. Follow menu instructions

---

## 🎯 Conclusion
This system models a real-world hospital environment using OOP principles and demonstrates how exception handling improves system reliability and user experience.
